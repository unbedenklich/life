# life

get ready to embark on an amazing journey through the beginnings of life.

start as a small cell and evolve...

relaxing short game (~10 minutes of playtime), spore-meets-agario, with a "our planet" documentary vibe.

made for gmtk jam 2024.

no graphical assets were used in this project, everything is procedurally generated.

CONTROLS:
A-D or moving mouse: rotate left/right

on mobile: tap somewhere to change direction.

reload to play again :P
